"Small Talk" Questions

What can we learn from this letter about the user experience?

Even a new, if not foreign, program should be learnable and manipulable.

How extendable was this system?

The system is pretty limited by what you can do as far as manipulating objects on screen, but Marian seemed to be able to use its full extent nonetheless.
As well as being able to create your own programs, which Marian did with her friend Colleen. The extendibility of their program requires that you know how to rotate, move, and scale polygons on screen; beyond that was illustration and design.

Points in which the user was on-boarded?

When she first started the program, she was shown how to load the disk into the computer and then load the file with the program on it.
She was taught how to create her own programs.
She was also taught how to make the polygons follow her cursor.
